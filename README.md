# mev-proxy

## Usage

Modify `MULTISIG_ADDRESS` and `IMPLEMENTATION_ADDRESS` in [deploy.py](./scripts/deploy.py)

```
ape accounts import alfa
> Enter pk
ape run scripts/deploy.py deploy --network <RPC_URL>
```
